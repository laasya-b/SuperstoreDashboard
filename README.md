# Superstore Sales Dashboard

This project predicts next month's sales for a Superstore dataset using Python and visualizes historical sales trends with an interactive dashboard built in Streamlit.

## Features
- Predicts next month’s sales using Linear Regression
- Visualizes historical sales trends and key insights
- Handles large datasets with preprocessing and feature engineering
- Interactive Streamlit dashboard for easy exploration

## Tech Stack
- Python: pandas, numpy, scikit-learn, matplotlib, seaborn
- Streamlit for the dashboard
